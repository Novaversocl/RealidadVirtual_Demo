# Virtual Reality Demo

This project is a virtual reality demonstration that uses Three.js for 3D graphics and TensorFlow.js along with MediaPipe for real-time hand tracking. The goal is to visualize key points of the hands in 3D and interact with 3D objects.
![Animation2](https://github.com/user-attachments/assets/f3a84529-fb13-4609-833e-c824cda16653)



## Features

- Real-time hand tracking using TensorFlow.js and MediaPipe.
- Visualization of key hand points in a 3D environment with Three.js.
- Interaction with 3D objects (e.g., moving a cube with the tip of the index finger).
